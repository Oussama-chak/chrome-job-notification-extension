# Job Notification Extension

A Chrome Extension that scrapes job listings from Indeed and sends notifications for new job offers.

## Setup

1. Clone this repository.
2. Go to `chrome://extensions` in Chrome.
3. Enable Developer Mode and click "Load unpacked."
4. Select the project folder to load the extension.

## Features

- Scrapes IT job listings from Indeed.
- Sends push notifications when new jobs match your profile.
